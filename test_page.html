Przykłady z niewidocznymi znakami

1. Zero-width space (U+200B)

Ten tekst zawiera​ niewidoczną spację o zerowej szerokości, która znajduje się pomiędzy słowami "zawiera" i "niewidoczną".

2. Zero-width non-joiner (U+200C)

Znak zero-width non-joiner jest używany głównie w językach, które używają ligatur, takich jak perski. Przykład: ن‌ه

3. Zero-width joiner (U+200D)

Zero-width joiner jest używany do tworzenia emoji z modyfikatorami. Przykład: 👨‍👩‍👧 (rodzina)

4. Różne niewidoczne znaki

Ten tekst zawiera​ różne niewidoczne znaki: spację zerowej szerokości, znak kierunkowy​ i inne niewidoczne​ znaki.

5. Ukryte hasło (celowo ukryte znaki)

Hasło: 123​456​789

Uwaga: W haśle powyżej znajdują się niewidoczne znaki, które mogą utrudnić jego skopiowanie!

6. URL z niewidocznymi znakami

Link: https:​//example.com

Uwaga: Link zawiera niewidoczny znak, który może przekierować na inną stronę niż się spodziewasz!

7. Soft hyphen (U+00AD)

Słowo z miękkim łącznikiem: super­kalifragilis­tyczny­ekspi­alidocjusz

8. Ukryte formaty tekstu z użyciem Left-to-right mark i Right-to-left mark

Ten tekst ma ‎ukryte‏ znaki kierunkowe, które mogą wpływać na formatowanie.

9. Różne rodzaje spacji

Normalna spacja: "a b"

Hair space (U+200A): "a b"

Narrow no-break space (U+202F): "a b"

Medium mathematical space (U+205F): "a b"

  • Nieoczekiwane znaki z edycji lub kopiowania tekstu z innych źródeł
  • Znaki specjalne dodane przez asystentów pisania lub narzędzia formatujące

Test 6: Złośliwe wykorzystanie niewidocznych znaków

Ten przykład pokazuje, jak niewidoczne znaki mogą być używane do ukrywania złośliwego kodu:

function loggedIn() {
  // Sprawdź uprawnienia użytkownika
  if (user.isAdmin​()) { // Uwaga! Ukryty znak!
    console.log("Użytkownik ma uprawnienia administratora");
    return true;
  }
  console.log("Użytkownik nie ma uprawnień administratora");
  return false;
}

W powyższym kodzie, funkcja isAdmin​() zawiera ukryty znak, który sprawia, że funkcja nigdy nie jest wywoływana! Zamiast tego, wyrażenie jest zawsze prawdziwe, co może prowadzić do nadania uprawnień nieautoryzowanym użytkownikom.

function loggedIn() { // Sprawdź uprawnienia użytkownika if (user.isAdmin​()) { // Uwaga! Ukryty znak! console.log("Użytkownik ma uprawnienia administratora"); return true; } console.log("Użytkownik nie ma uprawnień administratora"); return false; }
